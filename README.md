# NLE Cloud Collaboration Guide  
# 非线性剪辑软件云端协作入门指南  

## 简介 | Introduction  
这是一个面向新手的 **非线性剪辑（NLE）软件云端协作省流版教程合集**。  
内容涵盖 **DaVinci Resolve** 与 **Adobe Premiere Pro** 等主流视频后期软件的项目共享、版本管理和团队协作方法。  
教程基于个人学习与实践总结，仅供参考，欢迎讨论交流。  

This repository is a **beginner-friendly quick guide** to cloud-based collaboration in Non-Linear Editing (NLE) software.  
It covers practical tutorials for **DaVinci Resolve** and **Adobe Premiere Pro**, focusing on project sharing, version control, and teamwork workflows.  
All materials are based on personal understanding and experience — feel free to share your thoughts or improvements!
