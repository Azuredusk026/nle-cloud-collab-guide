# 主流剪辑软件云端协作教程合集 | Cloud Collaboration Guides for Mainstream Non-Linear Editing Software

## 写作初衷 / Purpose of This Repository

这个仓库收录了我个人编写的非线性剪辑（NLE）软件云协作使用教程，  
目前包含 **DaVinci Resolve** 与 **Adobe Team Projects** 的  
中英文版完整指南。

这些内容来自我的个人经验、官方文档及公开资料的整理与理解，  
目的在于：
- 帮助初学者更快上手云端协作功能；  
- 也帮助我自己在未来可以随时回看与继续学习。  

---

This repository collects guides on **cloud collaboration workflows**  
for Non-Linear Editing software.  
It currently includes full **English and Chinese** guides for  
**DaVinci Resolve Cloud Collaboration** and **Adobe Team Projects**.

All information is based on my personal experience, official resources,  
and publicly available materials.  
The goal is simple:
- To help beginners understand and start using these features faster;  
- And to serve as a personal reference for future study.
